How do we plan to solve the problem?

Manual analysis of financial reports is time-consuming, error-prone, and highly repetitive. The MAAB Talents team addresses this issue through a fully automated, accurate, and fast AI-powered platform.

1. We will implement the project through the following main stages:

Automatic data extraction from documents Read

Financial documents in PDF, Excel, and CSV formats are automatically converted into a structured format using AI.

Fine-tuning an LLM for FinQA Details

Compact yet efficient models (such as Llama 3 or Qwen 2.5) are fine-tuned on the FinQA dataset to accurately answer questions about numbers, tables, and formulas contained in financial reports.

AI-based analytical insights generation Open

Extracted data is passed to the model, and AI automatically generates key metrics, trends, and concise analytical insights.

Calculation of financial metrics Compute

If specific indicators are missing in the report or require manual computation (EBITDA, growth ratios, leverage, cash-flow forecasts), AI calculates them automatically.

Integrating everything into a unified web platform Platform

The user uploads a document → AI analyzes it → summaries, chatbot, visual charts, and PDF/Word/Excel export become available.
