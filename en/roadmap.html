Development strategy

Projects development stages — Idea → Prototype → MVP → Launched version

Our project is currently at the Idea stage

Within the hackathon, the concept of an AI-based platform for automatic financial report analysis has been formed, along with the core technical approaches.

Current step: Idea

  • •An extraction architecture for automatic reading of financial documents (PDF/Excel/CSV) has been developed.
  • •The concept of fine-tuning a small LLM on FinQA has been defined.
  • •A complete plan for analytics, automatic metric calculations, and web platform functionality has been created.
1

Prototype stage

  • •Building initial extraction modules capable of reading PDF, Excel, and CSV documents.
  • •Structuring and normalizing the data extracted from reports.
  • •Creating a demo question-answer interface for the AI model.
2

MVP (Minimum Viable Product)

  • •Initial fine-tuning of a small LLM on the FinQA dataset.
  • •Adding document upload, short summary generation, and chatbot features to the web interface.
  • •Launching the module for automatic calculation of key financial metrics.
  • •Exporting consolidated analysis into PDF/Word/Excel formats.
3

Launched version

  • •A full AI analytics module: trend detection, forecasts, advanced indicators.
  • •Extensive testing on real company reports.
  • •Deploying the improved UX/UI web platform on stable hosting.
  • •Introducing API integrations for corporate clients.
